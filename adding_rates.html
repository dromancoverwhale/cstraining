Solartis Builder: Adding Rates

Key Steps

1. Login to Solartis Builder
2. Click Rate Table
3. Search for Appropriate Rate Table
4. Check Off All Rate Tables That You Want to Update
5. Click Folder Icon Above Rate Table List, and Select Export
6. Add Values to All Applicable Fields (note: for date format should match '01/01/2024)
7. Do Not Add Values for CHANGE_VERSION, PRIORITY, CREATED_BY, CREATED_DATE, UPDATED_DATE, STATUS
8. Input "Add" as CHANGE TYPE Reason
