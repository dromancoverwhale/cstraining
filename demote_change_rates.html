Solartis Builder: Demoting and Updating Existing Rates

Notes

1. Changing Rate Values

Use this process to update existing factors (e.g., changing Masonry from 0.90 to 0.95).

  1. Select the required rate table and click Actions > Export to download the file.
  2. Open the file and update the rate values as needed.
  3. In the "Change Version" column, enter the value Change for every row you modified.
  4. Save the file and click Import to override the table.
  5. Verify the updates in the "Rate Table List" screen.

2. Deleting Rate Values

Use this process to remove specific rows from a live table.

  1. Export the required rate table.
  2. In the "Change Version" column, enter the value Delete against the rows you wish to remove.
  3. Save and Import the file.
  4. The rows marked "Delete" will be removed from the table upon import.

3. Demoting Rate Tables

Use this functionality to change the status of rate values from "Live" back to "Dev"].

  1. Export the required rate table.
  2. In the "Change Version" column, enter the value Demote against the required rows.
  3. Save and Import the file.
  4. Verify that the status has changed from "Live" to "Dev" in the Rate Table List.
